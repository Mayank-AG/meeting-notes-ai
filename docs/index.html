Quick start
🍎 macOS only  ·  Python 3 required (pre-installed on Mac)
1

Download & open Terminal in the folder

Download the ZIP above, extract it. Then open Terminal, drag the extracted folder into the Terminal window — it fills the path for you.

# Or type it manually — adjust path to where you extracted
$ cd ~/Downloads/meeting-notes-ai-main
2

Run setup (one time only · ~2 min)

Creates a virtual environment, installs dependencies, and asks for your API keys.

$ ./setup.sh
3

Start the app

Every time you want to use it — browser opens automatically at localhost:8000.

$ ./start.sh
🔒 All recordings, transcripts, and notes are saved to meeting-notes-ai/meetings/ on your Mac. Nothing is uploaded to any server — API calls go only to Sarvam AI (transcription) and Anthropic Claude (notes generation).

What you get
🇮🇳

Built for Indian meetings

Handles Hinglish natively — code-switching between Hindi and English mid-sentence, no problem.

✅

Exhaustive action items

Every "I'll check on this", "dekh lunga", "let's revisit" captured in a table. Pushed to Notion as individual rows you can filter by person.

🧠

Decisions with reasoning

Not just what was decided — but why. The full chain of reasoning, trade-offs, and analogies used in the meeting.

📂

Past meetings browser

Every meeting saved locally. Browse, re-read, or re-push any past meeting to Notion from inside the app.

⚙️

Team context

Tell it about your team, projects, and terms once. Notes become tailored to how you actually work.

📁

Upload audio files

Already have a recording? Drop in a .mp3 or .webm and run the full pipeline. Multiple files = auto-merged transcript.
