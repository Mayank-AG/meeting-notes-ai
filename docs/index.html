Get started on Mac
1

Download the ZIP

Click Download from GitHub above → Code → Download ZIP. Extract it anywhere — Downloads, Desktop, wherever.

2

Open Terminal · drag the folder in · run setup

Drag the extracted folder into your Terminal window — it fills the path automatically. Press Enter, then run ./setup.sh. Takes ~2 minutes.

3

Start the app

Run ./start.sh — browser opens automatically. Add your API keys in the Setup screen. Done.

🔒 All recordings, transcripts, and notes are saved to meeting-notes-ai/meetings/ on your computer. Nothing is uploaded to any server — API calls go only to Sarvam AI (transcription) and Anthropic (notes generation).

Windows? See the README for WSL and manual setup options.
