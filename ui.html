⚙️ Setup

🔊 Sarvam AI Required
Transcribes your recording. Built for Hinglish and code-switching. ~₹0.40 per minute of audio.
  1. Go to sarvam.ai and create an account (free)
  2. Click your profile → Dashboard
  3. Click API Keys in the sidebar
  4. Click Create New Key, give it any name, copy it
Cost: ~₹0.40 per minute of audio. A 1-hour meeting costs ~₹24.
🧠 Anthropic Claude Required
Generates your structured notes — decisions, action items, teaching moments. ~$0.003 per meeting (₹0.25).
  1. Go to console.anthropic.com
  2. Create an account or log in
  3. Click API Keys in the left sidebar
  4. Click Create Key, give it a name, copy the key (starts with sk-ant-api03-)
  5. Add a few dollars of credit under Billing — $5 covers ~1,600 meetings
Cost: ~$0.003 per meeting (~₹0.25). Essentially free for regular use.
📋 Notion Optional
Push notes + action items to Notion after every meeting. You create one database — a linked Action Items database is auto-created on your first push.
  1. Go to notion.so/my-integrations → New integration
  2. Give it a name (e.g. "Meeting Notes"), click Submit, copy the Internal Integration Token (starts with ntn_)
  3. In Notion, create a new full-page database (table view) — call it "Meeting Notes" or anything you like
  4. Open that database → click ⋯ (top right) → Connections → find your integration and connect it
  5. Copy the database ID from the URL: notion.so/workspace/THIS-LONG-ID?v=...
The Action Items database is created automatically on your first Notion push — you don't need to create it manually.
